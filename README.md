# FDESAWEB25

## Descripción del proyecto
Esta aplicación es un **gestor de contenidos personal** donde los usuarios pueden crear, organizar y compartir sus notas, artículos y recursos multimedia. La plataforma permite *categorizar contenido* de forma intuitiva.

## Público objetivo
- **Estudiantes universitarios** que necesitan apuntes.
- *Profesionalese* con múltiples proyectos.
- Cualquier persona que busque **organizar información personal**.

## Funcionalidades principales
- Crear y editar notas
- Categorización contenido con etiquetas
- Reportes semanales de productividad
- Sincronización entre dispositivos

## Tecnologías previstas
### Frontend
- **HTML5** - Estructura de la página
- **CSS3** - Estilos y diseño
- **JavaScript** - Interactividad y validaciones

### Backend
- **PHP** - Lógica del servidor
- **MySQL** - Base de datos

## Características clave (lista no numerada)
- ✅ Interfaz minimalista y fácil de usar
- ✅ Sincronización en tiempo real
- ✅ Modo oscuro/claro
- ✅ Exportación de datos a PDF/Excel
- ✅ Integración con Google Calendar

### Fases de desarrollo (lista numerada)
1. Investigación y planificación (2 semanas)
2. Diseño de prototipos en Figma (1 semana)
3. Desarrollo del frontend (3 semanas)
4. Implementación del backend (3 semanas)
5. Pruebas y despliegue (2 semanas)
